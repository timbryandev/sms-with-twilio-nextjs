# Next.js + Twilio

Example repo of how to send an SMS via Twilio using the NextJS stack

## Prerequisites

- Twilio account with a phone number set up
- Node>=14
- NPM >=6

## Setup

```bash
cp .env.example .env.local
# populate .env.local with your Twilio credentials
npm install
npm run dev
```
